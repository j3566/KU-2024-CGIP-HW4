KU-2024-CGIP-HW4
=============

- Coputer Graphics and Image Processing Homework 4

- 개발 환경
  - Windows / x86
  - Visual Studio 2022
  - GLUT 3.2
  - C++
  - CMake

- 빌드
  - 소스 경로에서 CMake 빌드
  - cmake -S . -B ./build -G "Visual Studio 17 2022" -A Win32
  - cmake --build ./build --config Release
  - ./build/Release/KU-2024-CGIP-HW4.exe 실행
